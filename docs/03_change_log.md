## Change Log 記載ルール
以後、仕様変更や方針変更があった場合は以下の形式で追記する。

### テンプレート
```md
## YYYY-MM-DD
- 変更内容
- 変更理由
- 影響範囲
```

### 記載例
```md
## 2026-03-20
- 日本語ページを公開対象から除外
- 理由: 内部確認用途のみで十分と判断したため
- 影響範囲: ルーティング、ナビゲーション、言語選択UI
```

## 運用メモ
